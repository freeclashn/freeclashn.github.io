一、Quantumult简介与基本概念

什么是Quantumult？

Quantumult是一款面向iOS平台的网络代理软件，支持多协议代理服务，帮助用户突破网络限制，保护上网隐私。其强大的自定义参数设定和丰富的功能模块，使其不仅适合普通用户，也满足高级用户的个性化需求。

基础概念解析

  • 代理（Proxy）：通过代理服务器转发网络请求，隐藏用户真实IP地址，绕过网络限制。

  • 规则（Rules）：用于判断哪些网络流量走代理，哪些流量直连，提升访问效率。

  • 配置文件（Configuration File）：保存代理服务器、规则、脚本等全部设定，是Quantumult运行的核心。

理解这几个基础概念，有助于您更好地掌握Quantumult的配置技巧。

三、Quantumult的基本参数设定

1. 代理服务器添加与管理

Quantumult支持多种代理类型，包括HTTP、HTTPS、SOCKS5等，用户可以灵活添加：

  • 路径：进入“设置” → “代理” → “添加代理”。

  • 配置代理信息：填写代理服务器地址、端口号、用户名和密码（如需）。

  • 选择代理类型：根据代理服务选择对应协议。

添加完成后，代理列表会自动更新，方便管理和切换。

2. 规则的设置与优化

规则决定哪些流量通过代理，哪些流量直连：

  • 添加规则：在“设置” → “规则”中添加自定义规则。

  • 规则格式：支持域名匹配、IP段匹配等多种格式。

  • 优先级管理：规则顺序影响匹配效果，优先级高的规则优先应用。

合理配置规则，既能保障访问速度，又能确保关键流量通过代理。

3. 配置文件的导入

配置文件集中保存所有代理和规则信息，导入方式多样：

  • 本地导入：在“设置” → “配置文件” → “导入”中选择本地文件。

  • 在线导入：通过配置文件URL直接导入，方便快捷。

导入后，Quantumult自动应用配置，极大简化设置流程。

四、Quantumult高级参数设定

1. 脚本的使用

Quantumult支持多种脚本语言，用户可以通过脚本实现高级功能：

  • 添加脚本：进入“设置” → “脚本”，导入或编写自定义脚本。

  • 常用脚本应用：实现广告屏蔽、智能分流、自动切换节点等。

  • 语言支持：JavaScript等主流脚本语言，扩展功能强大。

脚本极大提升了Quantumult的灵活性和自动化水平。

2. 订阅功能

订阅功能方便用户批量管理代理服务器：

  • 添加订阅：在“设置” → “订阅”中添加订阅链接。

  • 自动更新：设置自动刷新间隔，保证订阅节点及时更新。

  • 多订阅支持：可添加多个订阅，满足不同需求。

订阅使节点管理更便捷，维护代理稳定性更简单。

FAQ

V2Ray 的 gRPC 模式适合哪些场景？
gRPC 支持多路复用和长连接，适合高并发访问和移动网络场景。相比 TCP 或 WebSocket，它可以减少握手次数，降低延迟，同时提高连接稳定性和吞吐量，特别适合游戏或即时通讯。
Clash 节点延迟检测如何优化网络访问？
Clash 会定期测试节点延迟，并结合策略组自动选择最优节点。避免访问高延迟或失效节点，提升网页浏览、视频播放和游戏速度及稳定性，提高整体网络体验。
Clash 如何通过规则文件优化流量分配？
Clash 支持自定义规则文件，可根据域名、IP 或应用类型分流流量。结合策略组使用，可自动将视频、游戏、浏览器等流量分配到最优节点，提高访问速度和网络稳定性。
Shadowrocket 的日志界面怎么看节点状态？
Shadowrocket 的日志界面实时显示连接记录与节点响应延迟。绿色代表成功连接，红色代表失败，通过查看错误码可判断是 DNS 问题、TLS 握手失败还是代理超时。
Sing-box 支持按 IP 段分流吗？
支持。Sing-box 可根据 IP 段将国内流量直连，国外流量走代理，结合策略组实现智能分流。这样可保证访问速度和稳定性，同时优化节点使用效率，提高网络体验。
Nekobox 与 Hiddify Next 有什么区别？
Nekobox 更注重轻量化和本地化，界面简洁，支持多协议基础功能；而 Hiddify Next 则整合了订阅转换、测速、规则分流等高级功能，更适合技术用户使用。
Mihomo 的节点管理支持批量操作吗？
Mihomo 支持批量导入、删除和更新节点。通过策略组和分类管理，用户可以高效地管理数百个节点，同时进行测速和节点切换，提高操作效率。
VMess 的加密方式如何选择？
VMess 支持自动加密和 AEAD 加密等多种方式。AEAD 加密更安全且抗重放攻击，适合高安全性需求的网络环境，而普通加密在性能上稍快，但安全性略低。
Nekobox 可以自定义路由规则吗？
支持。用户可通过界面或配置文件定义路由规则，按域名、IP 或端口匹配不同节点，实现流量分流。自定义规则增强了网络访问灵活性和安全性。
Potatso Lite 的节点测速功能有何局限？
Potatso Lite 节点测速只能显示延迟，无法测出带宽或实际下载速度。此外，测速结果受网络波动影响较大，建议结合实际使用体验来判断节点性能。
